Some HTML Tags I know:

<a>
Anchor: defines a hyperlink, used to link from one page to another.
<article>
Article:Specifies independent, self-contained articles on the page.
<body>
Body: Define's the document or page's body.
<dl>
Descriptive List: A list of items, with descriptions... kind of like this one!
<img>
Image: embeds an image in an HTML page.
